Käivitage veebileht omal arvutil

Projekti käivitamiseks toimige järgmiselt:

1. Kui projekt on GitHubis juba olemas, saate selle kloonida käsuga: 

git clone https://github.com/Nikitorij/Programmeerimine.git

2.Installige sõltuvused: minge projekti kataloogi ja käivitage käsk:

cd countries-app
npm install

3.Käivitage projekt käsuga:

npm start

4. Pärast teie browseris avab aken lingiga http://localhost:3000 kus saab näha veebilehe.

Vajalikku riigi info vaatamiseks tuleb kirjutada riigi nimi otsingu lahtrisse ja vajutada nuppu "Search".

Veebilehel ilmub riikide nimekiri ja tulev vajutada sobivale.
